## 🛠️ How to Run the Programs
1. Clone this repository:
```bash
git clone https://github.com/JhanviMangtani/C-Programming.git
```
2. Navigate into the repo folder:
```bash
cd C-Programming
```
3. Compile and run a program:
```bash
gcc program.c -o program
./program
```
